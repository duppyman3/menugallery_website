# MenuGallery — website

Brochure site for **MenuGallery**: photograph a restaurant menu and see every dish as an appetizing image with prices and estimated calories, carbs, and protein. Also hosts the app's privacy policy and terms of service. Built with Next.js (App Router) and Tailwind CSS v4; deployed on Vercel.

```bash
npm run dev     # start the dev server at http://localhost:3000
npm run build   # production build
npm run start   # serve the production build
```
